What is this thing?

This app is a wrapper for the monthly Hacker News' Who is hiring? thread. It is not affiliated with YCombinator, I don't make a single cent for running it, etc. etc. I built it for myself, and then people found it useful so that's it.

Is the code on Github?

Sure! For both the website and the database. Check it out here. PRs are welcome.

How can I contact you?

PRs, bug reports, suggestions and so on all welcome. Drop me an email - and if you're into that kind of things, you can find my GPG key here.

What is the stack behind wwh? ?

Python + Flask + SQLAlchemy, Nginx (http + reverse proxy cache), MySQL, Pure CSS hosted on my Digital Ocean VPS. Kudos to the excellent BeautifulSoup library for web scraping.
